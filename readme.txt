"Ve Social Share Buttons" A plugin that will help to share your site on social network and help to increase more trafic on your website.

= Features =


* An options for show share buttons below/above of content area.
* Shortcode of share buttons
 
== Installation ==

Installing the plugin:

1. Upload the plugin files to the `/wp-content/plugins/ve-social-share-buttons` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings -> VE Social Share Buttons screen to configure the plugin
